# Query-Quote

Query-Quote is an EECS 767 project focused on retrieving movies from remembered dialogue. The current repo contains a polished project homepage, a submission-ready proposal draft, and the bundled transcript dataset used for planning the retrieval pipeline.

## Project Materials

- Proposal draft: `PROPOSAL.md`
- Homepage implementation: `app/page.tsx`
- Local dataset copy: `movie-transcripts-59k/`

## Development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Quality Checks

```bash
npm run lint
npm run build
```
